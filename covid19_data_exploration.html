Covid 19 Data Exploration

Data exploration of global COVID 19 dataset in SQL Server

  • View Project

In this project I have used SQL to do some data exploration on the COVID 19 data, the results of which will be visualised in Tableau in the next project. I have used the data on covid deaths (given in the table CovidDeaths) and covid vaccinations (given in the table CovidVaccinations), obtained from ourworldindata.org.

First I have looked at the number of deaths out of the total number of cases each day in India. I have only selected the data where continent is not a null value, since whenever it is given as a null, the corresponding value in the location column is replaced by the correct value for the continent, but the location column should contain the country names instead. The death percentage is calculated using (total cases/total deaths)*100. By the end of April 2021, we had 19,164,969 cases and 211,853 deaths. About 1.1% of the total population of our country died due to covid. So the likelihhood of a person dying if they contract covid is 1.1%.

Next I have tried to estimate the percentage of the population that got covid using (total cases/population)*100. It can be seen that by April 30 2021, about 1.4% of India's population was confirmed to have contracted the virus.

I tried to find which countries have the highest infection rate compared to their population. Andorra was found to have the highest infection rate, with about 17.1% of its total population been diagnosed with covid.

The next step was to check which country has the highest death count for its population. total_deaths column has nvarchar data type, so I had to first convert it to integer data type. We can see that the United States has had the highest death count of 576,232 deaths till the end of April 2021.

Next, I have looked at the total number of cases, total number of deaths and the death percentage per day globally. Since the new_deaths column has nvarchar data type, I first converted it to an integer. Starting with 98 total cases and 1 death on Jan 23 2020, we went on to have 879,014 total cases and 14,573 total deaths on April 30 2021. Also, the death percentage increased from 1% to 1.7% globally.

I have then tried to find out the total number of people in the world who have been vaccinated. I first joined the CovidDeaths and CovidVaccinations tables using an inner join on the location (country) and the date, and then I looked at the new vaccinations per day as well as the rolling count of the vaccinations, which is the cumulative sum of the new vaccinations for each particular country (using the partition by function).

Lastly, I wanted to get the percentage of the population that has been vaccinated per country using the rolling count RollingPeopleVaccinated. We can do this in two ways - using a CTE or using a temp table. In the first method, we have created a CTE PopvsVac, with the columns continent, location, date, population and the new_vaccinations each day, along with the cumulative count of the vaccinations RollingPeopleVaccinated. Then I have calculated the percentage of people who have been vaccinated in each country using (RollingPeopleVaccinated/Population)*100.

In the second method, I have created a temp table PercentPopulationVaccinated, with the columns continent, location, date, population, new_vaccinations and RollingPeopleVaccinated. Then I've again calculated the percentage of the vaccinated population using (RollingPeopleVaccinated/Population)*100.
